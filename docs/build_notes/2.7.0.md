# Build Notes - v2.7.0

## Summary
- Bumped crate version from `2.6.0` to `2.7.0`.
- Added matrix operation `Herm` (Hermitian / conjugate transpose):
  - Applies conjugate transpose to top matrix.
- Added matrix operation `MatPow` (integer matrix power):
  - Stack order: push matrix, then integer exponent, then apply `MatPow`.
  - Supports positive, zero, and negative integer exponents.
  - Negative powers use matrix inverse and fail for singular matrices.
- Wired both operations through:
  - backend calculator core (`src/lib.rs`)
  - API + wasm (`src/api.rs`)
  - matrix panel action buttons (`Post_Fix_Pro.html`)
- Updated help documentation for both operations.
- Added tests:
  - backend tests for Hermitian and MatPow behavior
  - API tests for Hermitian and MatPow endpoints
