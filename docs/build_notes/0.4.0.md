# Build Notes - v0.4.0

## Summary
- Bumped crate minor version from `0.3.0` to `0.4.0`.
- Added operation dispatch for scalar and complex arithmetic/scientific methods.
- Expanded tests to cover new operation behavior and state-preserving failure paths.
- Ran tests and release build.
- Copied `index.html` to `dist/index.html`.

## Commands Run
- `cargo test`
- `cargo build --release`
- `cp index.html dist/index.html`

## Build Result
- Tests: passed (`13` unit tests in `src/lib.rs`)
- Release build: success

## Backend Scope Added
- New `CalcError` variants:
  - `TypeMismatch`
  - `DomainError`
  - `DivideByZero`
- New engine methods:
  - Configuration: `set_angle_mode`
  - Binary ops: `add`, `sub`, `mul`, `div`
  - Unary ops: `sqrt`, `exp`, `ln`, `sin`, `cos`, `tan`
- New internal helpers:
  - Transactional op application (`apply_unary_op`, `apply_binary_op`)
  - Complex conversion and math helpers
- Guarantees:
  - Failed unary/binary operations leave stack unchanged.
