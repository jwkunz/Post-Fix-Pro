# Build Notes - v0.21.0

## Summary
- Bumped crate minor version from `0.20.0` to `0.21.0`.
- Added requested backend operators and wasm/API exports across categories:
  - Complex: `arg`, `abs`, `abs^2`, `conjugate`
  - Scientific: `inv`, `x^2`, `x^y`, `x root y`, `10^x`, `log10`, `e^x`, `ln x`, `2^x`, `log2`, `gamma`, `signum`
  - Trig: `atan2`, `to_rad`, `to_deg` (plus existing trig and inverse trig)
  - Numbers: `n!`, `nCr`, `nPr`, `mod`, `ran#`, `GCD`, `LCM`, `rnd`, `floor`, `ceil`, `decP`
- Added deterministic RNG state to calculator state for `ran#`.
- Expanded Rust unit tests for new operator coverage.
- Updated scalar keypad: shortened `Enter` and added `EXP` button beneath it.
- Added `hyp` UI modifier toggle to swap trig buttons between standard and hyperbolic variants.

## Commands Run
- `cargo test`
- `./scripts/package_dist_folder.sh`

## Build Result
- Unit tests: pass (`47` tests).
- WASM release build: success.
- Dist artifacts refreshed (`dist/index.html`, `dist/pkg`, `dist/wasm_base64.js`).

## Notes
- `decP` is implemented as decimal part: `x - trunc(x)`.
- `atan2` uses stack order `(y, x)` -> `atan2(y, x)`.
