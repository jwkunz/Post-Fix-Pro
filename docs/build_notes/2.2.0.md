# Build Notes - v2.2.0

## Summary
- Bumped crate version from `2.1.0` to `2.2.0`.
- Added matrix linear algebra operations:
  - `diag`: converts a vector matrix (`Nx1` / `1xN`) to a diagonal matrix.
  - `MatExp` (`mat_exp`): computes matrix exponential `e^A` for square matrices.
- Wired new operations through:
  - Rust backend calculator core (`src/lib.rs`)
  - API and wasm bindings (`src/api.rs`)
  - Matrix panel UI buttons and action map (`Post_Fix_Pro.html`)
- Updated help documentation (`help.md`) to include `diag` and `MatExp`.
- Added backend and API tests for both operations.
