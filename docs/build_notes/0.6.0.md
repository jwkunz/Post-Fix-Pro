# Build Notes - v0.6.0

## Summary
- Bumped crate minor version from `0.5.0` to `0.6.0`.
- Added matrix determinant, inverse, and linear solve operations.
- Added singular-matrix error handling and square-matrix validation paths.
- Expanded unit tests for linear algebra correctness and failure-state preservation.
- Ran tests and release build.
- Copied `index.html` to `dist/index.html`.

- `cargo test`
- `cargo build --release`

## Build Result
- Tests: passed (`25` unit tests in `src/lib.rs`)
- Release build: success

## Backend Scope Added
- New public methods:
  - `determinant` (matrix -> real)
  - `inverse` (matrix -> matrix)
  - `solve_ax_b` (matrix `A`, matrix `B` -> matrix `X`)
- New error variant:
  - `SingularMatrix`
- New internal helpers:
  - `matrix_determinant`
  - `matrix_inverse`
  - `matrix_solve`
  - `require_square`
- Numerical method details:
  - Partial-pivot Gaussian elimination
  - Gauss-Jordan elimination for inverse
  - Back-substitution for `solve_ax_b`
- Behavior guarantees:
  - Singular, type, and dimension failures keep stack state unchanged.
