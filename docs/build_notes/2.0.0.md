# Build Notes - v2.0.0

## Summary
- Bumped crate version from `1.1.0` to `2.0.0` for the breaking linear algebra roadmap.
- Added `nalgebra` as a backend dependency for new linear algebra operations.
- Added matrix/vector operations in Rust backend:
  - `dot` (vector dot product for `Nx1` / `1xN` matrix vectors)
  - `cross` (3-element vector cross product)
  - `trace` (square matrix trace)
  - `norm_p` (entry-wise p-norm for matrix/vector; stack order: matrix then `p`)
- Added matrix element-wise scalar binary support and matrix `conjugate` support with regression coverage.
- Wired new operations through:
  - `CalculatorApi`
  - wasm bindings
  - GUI matrix operations panel buttons and action map.
- Added/updated tests:
  - Backend tests for dot/cross/trace/norm_p.
  - API tests for dot/trace/norm_p.
  - Updated matrix scalar power assertion to use tolerance-based comparison.
