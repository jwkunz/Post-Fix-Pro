# Build Notes - v0.20.0

## Summary
- Bumped crate minor version from `0.19.0` to `0.20.0`.
- Reworked the GUI panel layout for clearer separation of concerns:
  - Stack moved into its own top-most panel.
  - Scalar keypad moved into its own panel with a 4-column layout.
  - Matrix builder/operations moved into a dedicated matrix panel.
  - Scientific functions moved into a dedicated scientific panel.
- Updated scalar keypad arrangement to:
  - `RAD, DEG, CE, C`
  - `Drop, Dup, Swap, Rot`
  - `+-, /, *, -`
  - `7, 8, 9, +`
  - `4, 5, 6, Enter`
  - `1, 2, 3, Enter`
  - `0` (wide), `.`, `Enter` (tall)
- Preserved existing JS-to-Rust action dispatch while rerouting moved buttons through panel-specific listeners.
