# Build Notes - v0.17.0

## Summary
- Bumped crate minor version from `0.16.0` to `0.17.0`.
- Improved responsive auto-scaling behavior for narrow screens and vertical phone layouts.
- Adjusted layout breakpoints so content shifts to one column sooner.
- Reduced control sizing pressure and collapsed multi-column utility sections on mobile.
- Ran tests and release build.
- Copied `index.html` to `dist/index.html`.

## Commands Run
- `cargo test`
- `cargo build --release`
- `cp index.html dist/index.html`

## Build Result
- Tests: passed (`41` tests total)

## UI Scaling Changes
- Main app width and grid now scale more smoothly and stack earlier (`<=1100px`).
- Buttons and entry text use `clamp(...)` sizing for width-adaptive scaling.
- Added narrow-screen layout collapse for:
  - matrix input grid
  - matrix action rows
  - matrix preset rows
  - complex input row
