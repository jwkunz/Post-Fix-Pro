# Build Notes - v0.26.0

## Summary
- Bumped crate minor version from `0.25.0` to `0.26.0`.
- Added switchable complex entry modes in the `Other` panel:
  - `a + bi` (Cartesian)
  - `mag + arg` (polar)
- Updated complex push behavior:
  - Cartesian mode pushes raw `(re, im)`.
  - Polar mode converts `(magnitude, argument)` into `(re, im)` before calling Rust.
  - Polar argument conversion respects current angle mode (`RAD` / `DEG`).
- Updated complex entry labels/placeholders/button text based on selected mode.
- Updated `help.md` to document the two complex entry modes.

- `./scripts/build_wasm.sh`

- WASM release build: success.
